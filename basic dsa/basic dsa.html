DSA Questions

1. What is a Stack?
A Stack is a linear data structure that follows the Last In First Out (LIFO) principle.
2. What is a Queue?
A Queue is a linear data structure that follows the First In First Out (FIFO) principle.
3. What is a Binary Tree?
A Binary Tree is a tree data structure where each node has at most two children.
4. What is a Linked List?
A Linked List is a linear collection of data elements, where each element points to the next one.
5. What is a Hash Table?
A Hash Table is a data structure that maps keys to values for highly efficient data retrieval.
